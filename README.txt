
ASSESSMENT 1 - EVENT CALENDAR WITH FULL CALENDAR PLUGIN


DESCRIPTION : 

User needs to select event date, event start time, event end time and enter event title and submit the details using SAVE EVENT button.

Details given by the user will be inserted in database and retrieved on the calendar.

Calendar shows the events saved by all the users on each date.
On clicking on the event, a popup appears which shows all the details of the particular event.
Given Weekly View, Daily View and Monthly View.


USED TOOLS : HTML, CSS, JavaScript, jQuery, FullCalendar Plugin, PHP & MySQL.



NOTE : Need to create database (DB NAME : fullcalendar) and import .sql file in WAMP PHP MyAdmin located in SQL-UPLOAD/fullcalendar.sql
